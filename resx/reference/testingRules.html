Правила тестирования

Вам будет предложено несколько вопросов закрытого типа. На каждый вопрос имеется несколько вариантов ответа.

Количество баллов, полученных за правильный ответ, варьируется и зависит от сложности вопроса:

  • простой - один балл
  • средний - два балла
  • сложный - три балла

За верный ответ выставляется максимальное количество баллов по данному вопросу, за неверный - ноль.
Ответ на вопрос с невыбранными вариантами ответа считается неверным.
Для перехода к следующему вопросу необходимо нажать на кнопку "ответить".
По окончании тестирования будет выведена статистика по текущему сеансу тестирования.
Статистика сохраняется в базу данных.
